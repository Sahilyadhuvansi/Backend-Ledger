## 🚀 Getting Started

### Quick Start (5 minutes)

```bash
# 1. Extract the archive
tar -xzf GenAI-Enhanced-Projects.tar.gz
cd enhanced-projects

# 2. Run installation script
chmod +x install.sh
./install.sh

# 3. Configure API keys in .env files
# See .env.example in each project

# 4. Start projects
# Follow instructions printed by install.sh
```

### Manual Setup

**Backend-Ledger:**
```bash
cd Backend-Ledger-AI-Enhanced/backend
npm install
cp .env.example .env
# Edit .env with your API keys
npm run dev
```

**POST_MUSIC:**
