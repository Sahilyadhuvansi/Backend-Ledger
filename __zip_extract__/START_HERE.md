## ⚡ Quick Start (5 Minutes)

### 1. Extract Archive
```bash
tar -xzf GenAI-Enhanced-Complete.tar.gz
cd enhanced-projects
```

### 2. Run Auto-Installer
```bash
chmod +x install.sh
./install.sh
```

### 3. Configure API Keys

Get free API keys:
- **Claude:** https://console.anthropic.com/ ($5 free credit)
- **OpenAI:** https://platform.openai.com/ ($5 free credit)
- **Google Cloud:** https://console.cloud.google.com/ ($300 free credit)
- **Pinecone:** https://app.pinecone.io/ (Free starter tier)

Add to `.env` files in each project.

### 4. Start Projects

**Terminal 1 - Backend-Ledger:**
```bash
cd Backend-Ledger-AI-Enhanced/backend
npm run dev
```

**Terminal 2 - Backend-Ledger Frontend:**
```bash
cd Backend-Ledger-AI-Enhanced/frontend
npm run dev
```

**Terminal 3 - POST_MUSIC:**
```bash
cd POST_MUSIC-AI-Enhanced/Backend
npm run dev
```

**Terminal 4 - POST_MUSIC Frontend:**
```bash
cd POST_MUSIC-AI-Enhanced/Frontend
npm run dev
```

**Terminal 5 - Portfolio:**
```bash
cd Sahilyadhuvansi-AI-Portfolio
npm run dev
```

### 5. Access Applications

- **Backend-Ledger:** http://localhost:5173
- **POST_MUSIC:** http://localhost:5174  
- **Portfolio:** http://localhost:3000
